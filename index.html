The Odin Project - Rock, Paper, Scissors challenge

This challenge is part of The Odin Projects JavaScript foundations course

The Challenge is to create a console based game that takes a user input (rock, paper, or scissors) and evaluates it against a randomly selected option for the computer

The winner should then be output.

The key requirements are:

  1. Start a new Git repo
  2. Create this HTML document and link it to the js file
  3. Create a 'counterPlay' function that returns Rock, Paper, or Scissors randomly
  4. Create a function that plays a single round, taking in parameters, and returing a string declaring the winner
  5. Results must be RETURNED and not console logged.
  6. Create a new function called game() that plays 5 rounds and announces the overall winner.
